 🚀 Galaxy Quest — Social Media Launch Campaign

Galaxy Quest is a new sci-fi adventure game.  
This project showcases a complete social media marketing campaign designed to promote its release.  

 🎯 Objective
To build awareness, engage the gaming community, and drive excitement for the upcoming launch of Galaxy Quest through creative visuals, captions, and hashtags.

 📱 Platforms Used
- Instagram  
- X (Twitter)  
- YouTube Shorts  
- Discord Community  

 🧰 Tools Used
- Canva (Designs and stories)
- Google Sheets (Content calendar)
- GitHub (Project presentation)
